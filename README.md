# react-typescript-boilerplate

目标是打造市面上最好的基于 React 脚手架

- []  完善的 TypeScript 类型定义
- [ ] 极致的构建体验（速度、体积）
- [ ] Tree Shaking
- [ ] 按需加载
- [ ] 支持约定式路由、配置式路由和命令式路由
- [ ] 合理的 Polyfill（babel、postcss）
- [ ] 丰富的 React 使用 examples
- [ ] 基于 Jest 的充分测试
- [ ] 开放的 ESLint 规则
- [ ] 基于 Prettier 和 StyleLint 的强制性格式化检查
- [ ] Git commit message validate
- [ ] 基于 husky 和 lint-staged 的代码提交前检查
